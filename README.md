# UK Housing Price Analysis

## Overview
This project analyses historical UK housing price data to understand long-term
price trends and market behaviour.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Insights
- UK house prices have increased significantly since the 1950s
- Rapid growth observed after the late 1990s
- Short-term price drops align with economic downturns
- Long-term upward trend remains strong

## Dataset
Historical UK housing price dataset (CSV format)
